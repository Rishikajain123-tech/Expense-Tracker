# 💰 Expense Tracker

A responsive web-based Expense Tracker built using **HTML, CSS, and JavaScript**. The application allows users to manage income and expenses efficiently with persistent data storage using the browser's Local Storage.

## 🚀 Live Demo

🔗 https://rishikajain123-tech.github.io/Expense-Tracker/

## 📂 GitHub Repository

🔗 https://github.com/Rishikajain123-tech/Expense-Tracker

---

## ✨ Features

- Add income and expense transactions
- Delete transactions
- Real-time balance calculation
- Track total income and total expenses
- Persistent data using Local Storage
- Responsive user interface

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## ⚙️ Installation

Clone the repository

```bash

```

Open the project folder

```bash
cd Expense-Tracker
```

Open `index.html` in your browser.
